### 2. **Защита маршрутов** ✅

#### Реализовано:
```typescript
// Базовая защита
<Route path="/dashboard">
  <ProtectedRoute>
    <DashboardV2 />
  </ProtectedRoute>
</Route>

// Защита по роли
<Route path="/center/crm">
  <ProtectedRoute allowedRoles={['center_admin', 'admin']}>
    <CenterCRM />
  </ProtectedRoute>
</Route>
```

| Тип страницы | Guest | User | Specialist | Center Admin | Admin |
