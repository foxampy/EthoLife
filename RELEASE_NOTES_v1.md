## 🚀 DEPLOYMENT

### Production Build
```bash
pnpm install --frozen-lockfile
pnpm run build
```

### Deploy Platforms
- ✅ **GitHub:** https://github.com/TimurSama/EthoLife
- ✅ **Render:** Auto-deploy on push
- ✅ **Vercel:** Ready for deployment

### Environment Variables Required
```env
SUPABASE_URL=
SUPABASE_SERVICE_KEY=
JWT_SECRET=
TELEGRAM_BOT_TOKEN=
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
NOWPAYMENTS_API_KEY=
```
